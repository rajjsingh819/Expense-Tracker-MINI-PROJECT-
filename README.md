# Expense-Tracker-MINI-PROJECT-

## PROBLEM STATEMENT

Create a console based Expense Tracker program in python that allows the user to record daily expenses and view summaries like total spending. Use only the concepts learned like
[loops, conditions, list, dictionaries and basis input/output)


## PROJECT DETAILS/ DESCRIPTION

You are required to build simple personal finance management tool.
The program should allow the user to:    


Add an expense with details like date, category, description, and amount.

View all recorded expenses in a clean format.

Exit the program gracefully when the user choose to.


All tasks must be implemented using loops, if-else, lists, and dictionaries only.
No user-defined functions or file handling should be used.
